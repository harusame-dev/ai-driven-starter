# a11y

アイコンのみのボタンなど、 視覚的にラベルが無い要素のテキスト代替には `aria-label` ではなく `sr-only` クラスを付けたテキスト要素を優先する。

## 振る舞いと見た目の一致

要素の振る舞いと見た目を一致させる。

- ページ遷移を行う場合は `a` / `Link` を使い、見た目もリンク（テキストリンクなど）にする
- 状態変更や副作用のある操作は `button` を使い、見た目もボタンにする
- 「リンクをボタンの見た目にする」「ボタンをテキストリンクの見た目にする」といった、振る舞いと外観の不一致は避ける
